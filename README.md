# roadmap-react

![Node.js](https://img.shields.io/badge/node->=24.0.0-brightgreen)

Простое React-приложение для визуализации roadmap изучения технологий.

## Требования

-   Node.JS v24

## Установка и запуск

1. Установите зависимости:

```
npm i
```

2. Запустите dev-сервер:

```
npm run dev
```

## Файлы roadmap

В папке `examples` есть пара тестовых roadmap. Также можно их взять [отсюда](https://github.com/kamranahmedse/developer-roadmap/tree/ed412ab0c2e4c7a14d5669d0d92fac287db1efcf/public/roadmap-content)
